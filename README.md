# 🔗 Supplier Merger

**供应商表格整合工具 · 按供应商编码智能合并 Excel 数据**

## 功能特性

| 功能 | 说明 |
|------|------|
| 🔗 自动匹配 | 按供应商编码精确匹配两张表的对应行 |
| 📊 字段拼接 | 单子金额、上货金额、原因用 `+` 连接（不求和） |
| 🔢 自动排序 | 结果按供应商编码升序排列 |
| 📐 格式保留 | 保留源表列宽、字体、对齐方式 |
| 📋 完整边框 | 所有单元格自动添加边框线 |

## 输入格式

两张 Excel 表需包含以下列：

| 列名 | 类型 | 说明 |
|------|------|------|
| 供应商编码 | 文本/数字 | 主键，用于两表匹配 |
| 供应商名称 | 文本 | 供应商名称 |
| 单子金额 | 数字 | 合并时用 `+` 拼接 |
| 上货金额 | 数字 | 合并时用 `+` 拼接 |
| 原因 | 文本 | 合并时用 `+` 拼接 |
| 开户名称 | 文本 | 保留表 1 数据 |
| 帐号 | 文本 | 保留表 1 数据 |
| 开户行信息 | 文本 | 保留表 1 数据 |

---

## 合并规则

| 场景 | 处理方式 |
|------|---------|
| 两表都有该供应商 | 金额/原因用 `+` 拼接（如 `100+200`） |
| 仅表 1 有 | 直接保留表 1 数据 |
| 仅表 2 有 | 直接保留表 2 数据 |
| 两表都没有 | 不出现在结果中 |

### 示例

**表 1：**
| 供应商编码 | 供应商名称 | 单子金额 |
|-----------|-----------|---------|
| 101001 | 供应商A | 100 |
| 101002 | 供应商B | 300 |

**表 2：**
| 供应商编码 | 供应商名称 | 单子金额 |
|-----------|-----------|---------|
| 101001 | 供应商A | 200 |
| 101003 | 供应商D | 500 |

**合并结果：**
| 供应商编码 | 供应商名称 | 单子金额 |
|-----------|-----------|---------|
| 101001 | 供应商A | 100+200 |
| 101002 | 供应商B | 300 |
| 101003 | 供应商D | 500 |
