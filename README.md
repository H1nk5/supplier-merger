# 供应商表格整合工具

根据供应商编码合并两张 Excel 表，将相同供应商的单子金额、上货金额、原因用 `+` 连接。

## 功能

- 按供应商编码匹配两张表
- 单子金额、上货金额、原因：用 `+` 连接（不求和）
- 按供应商编码升序排列
- 保留源表格式（列宽、字体、对齐）
- 所有单元格带边框

## 使用

### 运行 exe

双击 `供应商表格整合工具.exe`，选择两张表和输出路径，点击"开始整合"。

### 从源码运行

```bash
pip install pandas openpyxl
python main.py
```

### 打包 exe

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "供应商表格整合工具" main.py
```

## 输入要求

两张 Excel 表需包含以下列：

| 列名 | 说明 |
|------|------|
| 供应商编码 | 主键，用于匹配 |
| 供应商名称 | 供应商名称 |
| 单子金额 | 会用 + 连接 |
| 上货金额 | 会用 + 连接 |
| 原因 | 会用 + 连接 |
| 开户名称 | 保留表1数据 |
| 帐号 | 保留表1数据 |
| 开户行信息 | 保留表1数据 |

## 示例

表1：
| 供应商编码 | 供应商名称 | 单子金额 |
|-----------|-----------|---------|
| 101001 | 供应商A | 100 |

表2：
| 供应商编码 | 供应商名称 | 单子金额 |
|-----------|-----------|---------|
| 101001 | 供应商A | 200 |

整合结果：
| 供应商编码 | 供应商名称 | 单子金额 |
|-----------|-----------|---------|
| 101001 | 供应商A | 100+200 |
